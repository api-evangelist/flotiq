# Flotiq Plans

Flotiq offers four subscription tiers targeting individuals, growing projects, and enterprises. All paid plans include a 14-day free trial and support both monthly and annual billing (annual saves up to 15%).

## Free — $0/month

Designed for testing and small personal projects.

- 100,000 API calls/month (30-day rolling window)
- 1,000 content objects
- 20 content types
- 200 MB file quota
- 2 team members
- 1 webhook
- 99% uptime SLA
- Community support

## Basic — $37/month (or $444/year)

Recommended for live, actively growing projects.

- 300,000 API calls/month
- 10,000 content objects
- 50 content types
- 15 GB file quota
- 3 team members
- 20 webhooks
- 99.9% uptime SLA
- Best-effort support
- Additional seats: $12/month per seat

## Growth — $114/month (or $1,368/year)

For regulated environments and advanced production needs.

- 1,000,000 API calls/month
- 100,000 content objects
- Unlimited content types
- 500 GB file quota
- 6 team members
- 50 webhooks
- >99.99% uptime SLA
- 72-hour support response
- Additional seats: $12/month per seat

## Custom — Contact Sales

Enterprise solutions with flexible terms tailored to specific requirements.

- Custom API call quotas (reported up to 15M for advanced enterprise)
- Custom content object limits
- Custom file quotas
- Dedicated support SLAs
- Flexible billing arrangements

## Notes

- API call limits use a **30-day rolling window**, not a fixed calendar month
- Rate limits are tracked **per account**, not per individual API key
- All keys under an account share the same API call quota
- Payment accepted via Visa, MasterCard, American Express, Union Pay (via Stripe)
